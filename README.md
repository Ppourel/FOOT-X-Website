# 👟 FOOT X — Premium Footwear Website

Welcome to **FOOT X**, a modern and stylish footwear website designed to deliver a premium shopping experience. Built with clean UI, smooth interactions, and responsive design, this project showcases a complete front-end eCommerce layout.

---

## 🚀 Live Demo

🌐 **Visit Website:**
https://ppourel.github.io/FOOT-X-Website/

---

## ✨ Features

* 🛒 Interactive shopping UI (Add to cart animation)
* 🎯 Clean and modern design
* 📱 Fully responsive (Mobile + Desktop)
* 🔍 Product filtering UI
* 🔐 Login interface (UI only)
* 👤 About section with profile card
* 📬 Contact form section
* ⚡ Smooth scrolling & animations
* 🔔 Toast notifications for actions

---

## 🛠️ Built With

* HTML5
* CSS3 (Modern UI + Flexbox + Grid)
* JavaScript (Vanilla JS)
* Font Awesome Icons
* Google Fonts

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/foot-x.git
```

2. Open the folder:

```
cd foot-x
```

3. Run the project:

* Open `index.html` in your browser

## 🌍 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

1. Push code to GitHub
2. Go to **Settings → Pages**
3. Select branch: `main`
4. Save
